# React Tile Card


### Usage
```
<TileCard src={ image.png } title="React Tile Card">
	<CardText>
	 Content here.
	</CardText>
</TileCard>
```
### Options
children
React components, props are passed to children from TileCard

avatar
optional, a custom avatar to use in the header

noImageInHeader

name

src

title

subtitle

cardStyle

cardHeaderStyle

cardHeaderTitleStyle

onClick

cardWidth

cardHeight

tileWidth

tileHeight
