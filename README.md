# Material Tile


### Usage
```
<MaterialTile src={ image.png } title="React Tile Card">
	<CardText>
	 Content here.
	</CardText>
</MaterialTile>
```
### Options
children
child components components, props are passed to children from MaterialTile

customCard

avatar
optional, a custom avatar to use in the header

noImageInHeader

name

src

title

subtitle

cardStyle

cardHeaderStyle

cardHeaderTitleStyle

onClick

cardWidth

cardHeight

tileWidth

tileHeight
